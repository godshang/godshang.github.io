XML解析

Java在解析XML方面框架比较多，有通过DOM方式的，也有直接将XML映射到Java Bean的。DOM方式需要编写大量遍历DOM树、提取数据的代码，在XML文件变动或者新增XML文件时，需要做大量的编码工作，十分繁琐。从XML映射到Bean的方式则比较简单，通过XPath或者注解的方式，配置解析规则即可。下面简单比较了JAXB和Digester的性能。
