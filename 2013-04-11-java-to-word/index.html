Javaз”ҹжҲҗWordж–Үд»¶

еҸ‘иЎЁдәҺ 2013-04-11 еҲҶзұ»дәҺ жҠҖжңҜ иҜ„и®әж•°пјҡ

ж–№жЎҲжҜ”иҫғ

1гҖҒPOIпјҡApache POIеҢ…жӢ¬дёҖзі»еҲ—зҡ„APIпјҢе®ғеҸҜд»Ҙж“ҚдҪңOfficeзҡ„еҗ„з§Қж јејҸж–Үд»¶пјҢйҖҡиҝҮиҝҷдәӣAPIеңЁJavaдёӯиҜ»еҶҷExcelгҖҒWordзӯүж–Үд»¶гҖӮPOIеҜ№excelзҡ„еӨ„зҗҶеҫҲејәеӨ§пјҢдҪҶеҜ№дәҺwordзҡ„ж”ҜжҢҒдёҚжҳҜеҫҲеҘҪпјҢзӣ®еүҚеҸӘиғҪе®һзҺ°дёҖдәӣз®ҖеҚ•ж–Үд»¶зҡ„ж“ҚдҪңгҖӮ

2гҖҒдҪҝз”ЁXMLеҒҡжЁЎжқҝпјҡwordж”ҜжҢҒXMLж јејҸжҸҸиҝ°пјҢеӣ жӯӨеҲ©з”ЁVelicityзӯүжЁЎжқҝеј•ж“ҺпјҢе°ҶеҠЁжҖҒж•°жҚ®жёІжҹ“еҲ°XMLжЁЎжқҝдёӯпјҢд»ҺиҖҢз”ҹжҲҗwordж–ҮжЎЈгҖӮиҝҷз§Қж–№жі•жҜ”иҫғз®ҖеҚ•пјҢиҖҢдё”еӨ„зҗҶйҖҹеәҰеҫҲеҝ«пјҢж ·ејҸе’ҢеҶ…е®№зҡ„жҺ§еҲ¶д№ҹйқһеёёеҘҪпјҢжү“еҚ°зҡ„иҜқд№ҹдёҚиө°ж ·гҖӮеҸҜд»ҘиҜҙжҳҜз”ҹжҲҗWordж–ҮжЎЈзҡ„жңҖдҪіж–№жЎҲгҖӮ

дҪҝз”ЁXMLиҝӣиЎҢWordз”ҹжҲҗпјҢеҸҲжңүдёӨз§ҚдёҚеҗҢзҡ„ж–№жі•пјҡ

еңЁwordдёӯзј–иҫ‘еҘҪж ·ејҸпјҢеҸҰеӯҳдёәXMLж јејҸгҖӮз”Ёж–Үжң¬зј–иҫ‘еҷЁжү“ејҖиҝҷдёӘXMLж–ҮжЎЈпјҢдјҡеҸ‘зҺ°жҳҜдҪҝз”Ёж ҮеҮҶзҡ„XMLиҜӯиЁҖжҸҸиҝ°зҡ„ж–ҮжЎЈеҶ…е®№е’Ңж ·ејҸгҖӮд№ӢеҗҺе°ҶйңҖиҰҒеҠЁжҖҒжёІжҹ“ж•°жҚ®зҡ„дҪҚзҪ®пјҢдҪҝз”ЁжЁЎжқҝиҜӯиЁҖзҡ„еҚ дҪҚз¬ҰжҸҸиҝ°пјӣеҗҢж—¶иҝҳеҸҜд»ҘдҪҝз”ЁжЁЎжқҝиҜӯиЁҖдёӯзҡ„дёҖдәӣеҫӘзҺҜгҖҒжқЎд»¶зӯүжҺ§еҲ¶з»“жһ„пјҢжҜ”еҰӮиҝӯд»Јз”ҹжҲҗиЎЁж јзҡ„иЎҢгҖӮз»ҸиҝҮиҝҷж ·еӨ„зҗҶеҗҺзҡ„XMLж–ҮжЎЈпјҢе°ұеҸҜд»ҘдҪңдёәжЁЎжқҝдҪҝз”ЁдәҶгҖӮеҗҢдҪҝз”Ёе…¶д»–жЁЎжқҝдёҖж ·пјҢе°ҶеҠЁжҖҒж•°жҚ®жү“е…ҘжЁЎжқҝеҗҺпјҢжүҖеҫ—еҲ°зҡ„ж–Үжң¬жөҒеҶҷе…ҘеҲ°еҸҰдёҖдёӘж–Үд»¶дёӯпјҢе№¶еҸҰеӯҳдёәdocж јејҸпјҢиҝҷж ·е°ұеҫ—еҲ°дәҶз”ҹжҲҗеҗҺзҡ„wordж–ҮжЎЈгҖӮ

иҝҷз§Қж–№жі•жңүдёҖдёӘй—®йўҳпјҢе°ұжҳҜжңҖеҗҺеҫ—еҲ°зҡ„ж–ҮжЎЈпјҢиҷҪз„¶иў«еҸҰеӯҳдёәdocж јејҸпјҢд№ҹиғҪеӨҹиў«office wordжӯЈеёёзҡ„жү“ејҖе’Ңзј–иҫ‘пјҢдҪҶеҰӮжһңз”Ёж–Үжң¬зј–иҫ‘еҷЁжү“ејҖпјҢдјҡеҸ‘зҺ°е®ғдҫқз„¶жҳҜдёӘXMLж–ҮжЎЈпјҢеә•еұӮзҡ„ж•°жҚ®еӯҳеӮЁж–№ејҸе№¶жңӘеҸ‘з”ҹеҸҳеҢ–гҖӮиҝҷеҜ№дәҺеҸӘз”ҹжҲҗгҖҒзј–иҫ‘е’Ңжү“еҚ°wordжқҘиҜҙе№¶дёҚжҳҜдёӘй—®йўҳпјҢиҝҷз§Қж–№жі•е®Ңе…Ёж»Ўи¶іиҝҷдәӣйңҖжұӮгҖӮдҪҶеҰӮжһңиҝӣдёҖжӯҘпјҢжғідҪҝз”ЁOpenOfficeе°ҶиҝҷдёӘwordиҪ¬дёәPDFзҡ„иҜқпјҢе°ұдјҡеҮәзҺ°й—®йўҳдәҶгҖӮдҪ дјҡеҸ‘зҺ°пјҢиҪ¬жҚўеҗҺзҡ„PDFпјҢйЎөйқўе…ЁжҳҜXMLж ҮзӯҫгҖӮгҖӮ

еӣ жӯӨпјҢеҰӮжһңжғіеҫ—еҲ°вҖңзңҹзңҹжӯЈжӯЈвҖқзҡ„wordж–ҮжЎЈжқҘиҜҙпјҢе°ұеҫ—дҪҝз”Ёз¬¬дәҢз§Қж–№жі•дәҶгҖӮз¬¬дәҢз§Қж–№жі•е®Ңе…Ёи§ЈеҶідәҶиҝҷдёӘй—®йўҳпјҢдҪҶеҗҢж—¶еҸҲжңүеҸҰдёҖдёӘеұҖйҷҗпјҢе®ғеҸӘиғҪй’ҲеҜ№word2007ж јејҸпјҢеҚіdocxиҝҷз§Қж јејҸгҖӮдёҚиҝҮзӣ®еүҚжқҘиҜҙпјҢoffice 2003иҷҪз„¶жІЎе®Ңе…Ёж·ҳжұ°пјҢдҪҶ2007гҖҒ2010зҡ„дҪҝз”ЁзҺҮд№ҹжҳҜз»қеӨ§еӨҡж•°гҖӮиҝҷдёӘй—®йўҳе°ұеҪ“е®ғдёҚжҳҜдёӘй—®йўҳеҘҪдәҶгҖӮгҖӮ

д»Ӣз»Қз¬¬дәҢз§Қж–№жі•еүҚпјҢйҰ–е…ҲжқҘдәҶи§Јword2007ж–ҮжЎЈзҡ„ж јејҸгҖӮWord2007ж–ҮжЎЈжң¬иә«жҳҜд»ҘXMLж–Үд»¶ж јејҸеӯҳеӮЁзҡ„пјҢз„¶еҗҺе°Ҷзӣёе…ізҡ„XMLж–Үд»¶е’Ңиө„жәҗж–Үд»¶еҺӢзј©еҗҺеҫ—еҲ°гҖӮдҪ еҰӮжһңе°ҶдёҖдёӘе·Із»Ҹеҗ«жңүеҶ…е®№зҡ„word 2007зҡ„ж–ҮжЎЈзҡ„жү©еұ•еҗҚз”ұdocxж”№дёәzipпјҢз„¶еҗҺдҪ дјҡеҸ‘зҺ°иҝҷдёӘеҺӢзј©еҢ…дёӢзҡ„зӣ®еҪ•з»“жһ„зұ»дјјиҝҷж ·пјҡ

зӣ®еҪ•з»“жһ„жҜ”иҫғеӨҚжқӮпјҢдҪҶд»ҺеҗҚеӯ—д№ҹиғҪеӨҹеӨ§жҰӮзҢңеҮәжқҘдёҖдәӣпјҢdocument.xmlдёӯеӯҳеӮЁзҡ„жҳҜйЎөйқўдёӯзҡ„еҶ…е®№пјҢheader1.xmlеӯҳеӮЁзҡ„жҳҜйЎөзңүзҡ„еҶ…е®№гҖӮгҖӮгҖӮз”Ёж–Үжң¬зј–иҫ‘еҷЁжү“ејҖиҝҷеҮ дёӘж–Үд»¶пјҢд№ҹйӘҢиҜҒдәҶзҢңжғігҖӮ

еңЁдәҶи§ЈдәҶword2007ж–ҮжЎЈзҡ„ж јејҸеҗҺпјҢд№ҹе°ұжңүдәҶи§ЈеҶізҡ„ж–№жЎҲгҖӮжһ„е»әword2007ж–ҮжЎЈзҡ„й—®йўҳд№ҹе°ұиҪ¬еҢ–жҲҗдәҶдҝ®ж”№дёҖдёӘеҺӢзј©ж–Үд»¶дёӯзҡ„xmlй—®йўҳгҖӮеҗҢж ·еҸҜд»ҘдҪҝз”ЁжЁЎжқҝжҠҖжңҜжқҘеҗ‘XMLж–ҮжЎЈдёӯеҠЁжҖҒзҡ„жү“е…Ҙж•°жҚ®пјҢз„¶еҗҺжӣҝжҚўеҺӢзј©ж–Үд»¶дёӯзҡ„зӣёеә”XMLж–ҮжЎЈпјҢе°ұеҸҜд»Ҙе®һзҺ°з”ҹжҲҗword 2007зҡ„йңҖжұӮгҖӮ

жҠҖжңҜе®һзҺ°

д»ҘдёӢз”ЁдёҖдёӘз®ҖеҚ•зҡ„е®һдҫӢд»Ӣз»Қе…·дҪ“зҡ„е®һзҺ°жӯҘйӘӨгҖӮ

з¬¬дёҖжӯҘпјҡеҲӣе»әжЁЎжқҝ

йҰ–е…ҲпјҢж–°е»әдёҖдёӘword2007ж–ҮжЎЈ,е№¶е®ҢжҲҗеҶ…е®№е’Ңж ·ејҸзҡ„зј–иҫ‘гҖӮз„¶еҗҺе°ҶиҝҷдёӘdocxж–ҮжЎЈзҡ„жү©еұ•еҗҚж”№дёәzipпјҢдҪҝз”ЁеҺӢзј©е·Ҙе…·жү“ејҖпјҢе°Ҷе…¶дёӯзҡ„document.xmlи§ЈеҺӢеҮәжқҘпјҢз„¶еҗҺд»ҺеҺӢзј©ж–Үд»¶дёӯеҲ йҷӨdocument.xmlгҖӮз„¶еҗҺе°ҶеҺӢзј©ж–Үд»¶еҗҚдҝ®ж”№дёәtemplate.docxгҖӮ

з¬¬дәҢжӯҘпјҡзј–еҶҷд»Јз Ғ

  1. е°Ҷdocument.xmlдёӯдҪҝз”ЁжЁЎжқҝиҜӯиЁҖиҝӣиЎҢйў„еӨ„зҗҶгҖӮ
  2. дҪҝз”ЁжЁЎжқҝеј•ж“Һе°Ҷж•°жҚ®еҠ иҪҪеҲ°document.xmlж–Үд»¶дёӯгҖӮ
  3. дҪҝз”ЁJavaдёӯзҡ„ZipInputStream/ZipOutputStreamж“Қзәөzipж–ҮжЎЈпјҢе°ҶдёҠдёҖжӯҘз”ҹжҲҗзҡ„document.xmlеҺӢзј©еҲ°docxдёӯгҖӮ
  4. жңҖеҗҺиҫ“еҮәдёәдёҖдёӘж–°зҡ„docxж–Үд»¶гҖӮ

иҝҷз§Қж–№жі•зҡ„е…ій”®зӮ№е°ұжҳҜиҝҷж ·гҖӮж–№жі•жң¬иә«иҝҳжҳҜеҫҲеҘҪзҗҶи§ЈпјҢйә»зғҰзҡ„еңЁдәҺжЁЎжқҝзҡ„еӨ„зҗҶдёҠгҖӮиҝҷйҮҢиҝҳжІЎеҸҚеә”еҮәиҝҷз§Қйә»зғҰпјҢеӣ дёәеҸӘжҳҜз”ҹжҲҗдәҶwordж–ҮжЎЈиҖҢе·ІпјҢзӯүеҲ°иҝӣиЎҢиҪ¬жҚўpdfзҡ„ж—¶еҖҷпјҢиҝҳдјҡжңүж–°зҡ„й—®йўҳеҮәзҺ°гҖӮгҖӮгҖӮ
